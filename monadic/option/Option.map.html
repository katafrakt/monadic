    Function Option.map

    Calls fun on value of Some or returns self if called on None. This is used to create chains of transformations, which might be stopped at any point when None is returned.

    Option map(alias fun)();

    Returns

    Option

    Example

    map on Some 

    auto some = Some(4);
auto doubleMe = (int x) => x * 2;
auto result = some.map!doubleMe;
assert(!result.isNone());
assert(result.unwrap!int() == 8);

    Example

    map with lambda 

    auto some = Some(4);
auto result = some.map!((int a) => a * 3);
assert(!result.isNone());
assert(result.unwrap!int() == 12);

    Example

    map on None 

    auto none = None();
auto doubleMe = (int x) => x * 2;
auto result = none.map!doubleMe;
assert(result.isNone());
