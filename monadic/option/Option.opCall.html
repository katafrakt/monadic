    Function Option.opCall

    Wraps a value in an Option. If the value is null, returns None. Otherwise returns Some(value)

    auto opCall(S) (
      S value
    );

    Returns

    None or Some(value)

    Example

    Wraping a value 

    auto some = Option(1);
assert(is(typeof(some) == Some));
assert(!some.isNone());

    Example

    Wrapping null 

    auto some = Option(null);
assert(is(typeof(some) == None));
assert(some.isNone());
