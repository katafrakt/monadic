    Function unwrap

    Takes a value out of Some and casts it into a T type. Raises CannotUnwrapNone exception if called on None.

    T unwrap(T) (
      Option option
    );

    Returns

    value of type T

    Throws

    CannotUnwrapNone if called on None, VariantException if cannot be casted to T

    Example

    auto someInt = Some(13);
assert(someInt.unwrap!int == 13);
assert(someInt.unwrap!long == 13);
assert(someInt.unwrap!double == 13.0);

    Example

    import std.exception : assertThrown;
import std.variant : VariantException;

auto someStr = Some("abc");
assertThrown!VariantException(someStr.unwrap!int());

    Example

    import std.exception : assertThrown;

auto none = None();
assertThrown!CannotUnwrapNone(none.unwrap!int());
